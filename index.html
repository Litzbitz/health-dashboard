睡眠统计逻辑修改 修改睡眠统计与分析逻辑： 1. 睡眠归属规则 不要按自然日（0:00-24:00）切割睡眠。 改为： 所有睡眠段归属于“醒来的日期（wake date）”。 例如： * 6月24日 22:00 → 6月25日 05:00 归属 6月25日 * 6月25日 22:00 → 6月26日 05:00 归属 6月26日 2. 每日总睡眠 同一天的多段睡眠需要自动累加： 例如： 6月25日： * 02:00-05:00 * 14:00-16:00 则： 每日总睡眠 = 5小时 3. 所有分析统一使用每日总睡眠 以下模块全部基于“醒来日期对应的总睡眠时长”： * 平均睡眠时长 * 睡眠评级 * 睡眠趋势 * 睡眠与头痛相关性 * 睡眠与肩颈痛相关性 * 睡眠与经期相关性 * 风险预测 不要使用自然日切割后的睡眠数据。 4. 睡眠评级（科学版） 基于每日总睡眠： * ≥ 9h → 偏长 * 7–9h → 充足 * 6–7h → 略不足 * 4–6h → 不足 * < 4h → 严重不足 同时显示： * 平均睡眠时长 * 睡眠评级 * 最近7天评级
